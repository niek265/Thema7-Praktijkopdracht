## Data explanation

The data follows the following convention:  
FlyLine_Age_Treatment_Replicant

Fly lines:
* Elav x 51D
* Elav x PrP (Cyt)
* Elav x PrP (GPI)

Age in days:
* 5
* 30
* 40

Treatment:
* Normal Brain Homogenate
* Scrapie

All data was merged into [GSE144028.txt](GSE144028.txt) using [data-loading.r](data-loading.r).
